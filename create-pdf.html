Create a PDF file in Python

Learn how to create a PDF file using Python.

      
from reportlab.pdfgen import canvas

# Create a new PDF file

pdf_file = canvas.Canvas("Example.pdf")

# Add text to the PDF file

pdf_file.drawString(72, 720, "Hello, Python!")
pdf_file.drawString(72, 700, "Free PDF File")
pdf_file.drawString(72, 680, "companionemlinguae.info")

# Save the PDF file

pdf_file.save()
